You use data binding to specify things such as

  • [src] - the source of an image
  • [disabled] - the state of a button
  • [name] - data for a particular user

HTML attributes and DOM properties

Angular binding distinguishes between

  • HTML attributes are static string values
  • when DOM is loaded, DOM properties are initialized with static Attribute string values
  • samples: class="selected" , button disabled img src="imgs/img1"
  • In Angular, the only role of HTML attributes is to initialize element and directive state
  • DOM properties are features of DOM nodes and are dynamic values
  • with data binding, you're dealing exclusively with the DOM properties and events
